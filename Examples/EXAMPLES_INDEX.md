# Examples Index

All non-runtime examples and sample assets are centralized here.

## Folders

- `Examples/Excel`
  - Sample spreadsheets, CSV files, and invoice PDFs used for imports and local validation.
- `Examples/Samples`
  - OCR parser validation inputs and generated reports.
- `Examples/Templates`
  - Environment and Apache template files (`.env.example`, `.htaccess.example`).

## Notes

- These files are for reference, testing, and setup only.
- Webapp runtime files remain at project root and active app directories.
